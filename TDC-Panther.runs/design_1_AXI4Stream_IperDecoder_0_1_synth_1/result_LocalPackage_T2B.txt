======================================================
n in 0 to N-1 stages
------------------------ Init ------------------------
NIT[0]		is set by Generic
MIV[0]		is set by Generic
LNG_IN[0]	 = ceil(log2(MIV[0])+1)
------------------------------------------------------
---------------------- Computation -------------------
NIE[n]		is set by Generic
NIT[n]		= ceil(NIT[n-1]/NIE[n-1])
MIV[n]		= MIV[n-1]*NIE[n-1]
LNG_IN[n]	 = ceil(log2(MIV[n])+1)
LNG_OUT[n-1]	 = LNG_IN[n]
------------------------------------------------------
------------------------ Last ------------------------
LNG_OUT[N-1]	= LNG_IN[N-1+1]
		= ceil(log2(MIV[N])+1)
		= ceil(log2(ceil(NIT[N-1]/NIE[N-1]))+1)
------------------------------------------------------
